嵌入图片

1.img元素

元素 作用
img 表示在html文档里嵌入图像

img元素的一些常用属性：

属性 作用
src 图片地址（链接or路径）
width 设置图片宽度
height 设置图片长度
alt 定义img元素的备用内容（浏览器找不到图片资源时起作用）

3.创建客户端分区响应图

原理：通过点击某张图像上的不同区域让浏览器到不同的url上

元素 作用
map 客户端分区元素，area写在map元素内部
area 每个area元素各自代表一个区域

1.area元素

area元素属性：

1.第一类属性：点击后导航到指定url

属性 作用
href 该区域被点击时浏览器应该加载的url
alt 定义图片的备用内容

2.第二类属性：shape属性和coords属性

两种属性同时起作用，标明用户可以点击的各个图像区域

shape coords属性
rect 四个整数（左上角x坐标，左上角y坐标，右下角x，右下角y）
circle 三个整数（圆心x坐标，圆心y坐标，圆半径）
poly 代表一个多边形，至少包含六个逗号分割的整数，每一对数字代表一个顶点坐标
default 代表默认区域，即整个图片，不需要coords
