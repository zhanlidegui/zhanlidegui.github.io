2.video元素属性

属性 作用
src 视频地址
width 设置视频所占宽度
height 设置视频所占长度
autoplay 网页完成后自动播放视频
preload 预先载入视频
controls 声明该属性，浏览器添加播放控件
loop 视频循环播放
poster 视频载入时显示图片
muted 视频静音

preload属性：

值 作用
none 不会自动载入
metadata 载入第一帧
auto 全部自动载入（不声明则默认为auto）

2.浏览器支持的视频格式

格式 ie Firefox chrome Opera Safari
ogv（ogg） 不支持 3.5以上支持 5.0以上 10.5 不支持
MP4 9.0以上支持 不支持 5.0 不支持 3.0
webm 不支持 4.0以上支持 6.0 10.6 不支持

ogg=带有theora视频编码和vorbis音频编码的ogg文件

MP4=带有H.264视频编码和aac音频编码的MPEG4文件

webM带有vp8视频编码和vorbis音频编码的webM文件

总结：1.没有任何一个格式可以支持所有浏览器；

​ 2.最简单的方法，需要MP4与ogv两种影音格式；

​ 3.现在更新即时可以支持了，但以前旧版本不支持，所以还需要添加。

所以，我们需要分类讨论，不同浏览器要用不同格式。
